Ghost leg est une méthode bien connue pour obtenir des résultats aléatoires. Ce mécanisme a pour effet de modifier l'ordre de la liste.

Cette mission est presque l'inverse de la mission Follow Ghost Legs. Mais cette mission a un nombre incalculable de réponses et une difficulté asymétrique. Vous devez donc résoudre le Follow Ghost Legs en premier.

On vous donne comme valeurs d'entrée une liste de résultats pour le site Ghost leg. La valeur de départ est toujours numérotée séquentiellement à partir de 1. La valeur de départ est toujours numérotée séquentiellement à partir de 1. Vous devez retourner la liste des jambes qui correspondent.

  • Le segment est un tuple de deux entiers, qui représentent les deux lignes verticales connectées par le segment. (Le segment relie toujours une ligne verticale adjacente).
  • L'ordre des pattes représente le niveau auquel elles sont placées (la première patte est en haut, et la dernière patte est en bas).

Votre réponse sera confirmée par le vérificateur.

Exemples: 

def checker(func: callable, result: list[int]):
    your_result = sorted(result)
    for a, b in func(result[:]):
        your_result[a - 1], your_result[b - 1] = your_result[b - 1], your_result[a - 1]
    return your_result == result

Entrée: Un résultat de 'Ghost Leg' (une liste d'entiers).

Sortie: une liste de pattes (une liste de tuple de deux entiers): Une liste de jambes (une liste de tuple de deux entiers).
